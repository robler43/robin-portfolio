# Robin Hoesli Portfolio

A minimalist UX Designer portfolio website with web3 aesthetic.

## Design Features

- **Pure black background** with teal (rgb(37, 208, 171)) accents
- **Space Mono monospace font** for authentic hacker aesthetic
- **Dot grid pattern** background
- **Clean monochrome** zinc/gray tones
- **Responsive design** for all devices

## Sections

- **Navigation**: Home, About, Work
- **Hero**: "athlete. designer. builder." with professional tagline
- **Project Grid**: 3 featured case studies with tags
- **Footer**: Copyright information

## Tech Stack

- React
- Tailwind CSS
- TypeScript
